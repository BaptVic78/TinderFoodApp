# Founder PWA (starter)

Une mini PWA installable iOS/Android.

## Démo locale
1. Ouvre un terminal dans ce dossier et lance un serveur local :
   ```bash
   python -m http.server 8000
   ```
2. Va sur http://localhost:8000
3. La PWA s'installe via Chrome (Android) ou Safari (iOS: Partager → Ajouter à l’écran d’accueil après déploiement HTTPS).

## Déploiement rapide (gratuit)
- **GitHub Pages** :
  - Crée un repo, uploade tous les fichiers à la racine.
  - Settings → Pages → Deploy from a branch → branch `main` / folder `/root`.
  - Ouvre l'URL fournie, puis installe la PWA.

- **Vercel/Netlify** :
  - Import du repo GitHub, déploiement en 1 clic.
